Build on macOS with the Apple command-line tools installed:

    xcrun clang -Os \
      fan_speed_probe.c smc.c \
      -framework IOKit \
      -framework CoreFoundation \
      -o FanSpeedProbe

Mac Unseen invokes this program as a separate process and reads its JSON
standard output. No signing key or non-public build tool is required to
compile or replace it.
